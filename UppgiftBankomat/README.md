## Mjukvaruprogram info
Bankomat-programmet är en konsolapplikation som simulerar en bankomat. Programmet börjar i Program-klassens Main-metod
och därifrån skapar ett Bankomat-objekt som innehåller huvudmenyn för användaren. Användaren kan genom huvudmenyn:
1.  Gör en insättning på ett konto
2. Gör ett uttag på ett konto
3. Visa saldot på ett konto
4. Skriv ut en lista på alla kontonr och saldon
5. Avsluta programmet
